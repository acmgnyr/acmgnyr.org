How to read the table:

In columns A through I, the entries look like number / time. The number is the number of submissions; the time is the total time penalty for that problem. If the time is --, it means that the team did not solve that problem.

In the Total column, the entries look like correct / total. The correct is the number of correct submissions, and total is the total number of submissions.

In the last row, the entries look like attempt / correct / time. The attempt is the number of submissions for that problem; correct is the number of correct submissions. The time is the penalty for the first submission.
