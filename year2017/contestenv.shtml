2017 Contest Environment Information

You need to boot into the GNYR contest environment in order to compete. It provides a programming environment, contest software, and connection to our contest server to compete.

Things you need before you begin:

  • One (or two) USB keys at least 4 GB in size.
  • A wired connection to the Internet. (Not wireless.)

Then, choose your OS:

  • Linux
  • Windows
  • Mac OS / OS X
