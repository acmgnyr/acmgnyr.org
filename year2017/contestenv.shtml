2017 Contest Environment Information

You will need to boot your computer into the GNYR contest environment (by a USB drive or DVD) in order to compete in this year's GNYR Contest. The environment provides programming tools, contest software, and connection to our contest server to compete.

Provided here are instructions how to create that USB drive and what to do once you boot into it. Things you need before you begin:

  • One (or two) USB keys at least 4 GB in size.
  • A wired connection to the Internet. (Not wireless.)

Then, choose the OS you are using to create a contest environment USB drive:

  • Linux
  • Windows
  • Mac OS / OS X
  • Bootable DVD (last resort)
